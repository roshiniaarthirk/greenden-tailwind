🌿 Greenden - Tailwind CSS Project
Greenden is a responsive, modern e-commerce landing page for a plant nursery/store. This project focuses on clean UI, aesthetic plant displays, and a seamless user experience using Tailwind CSS.

🚀 Features
Responsive Design: Fully optimized for Mobile, Tablet, and Desktop.

Modern UI: Clean and green-themed interface.

Search Functionality: Functional search bar for plant products.

Product Categories: Organized sections for different plant types.

Interactive Components: Hover effects and smooth transitions using Tailwind utilities.

🛠️ Tech Stack
HTML5: Semantic structure.

Tailwind CSS: For rapid and modern styling.

JavaScript: (Optional) For basic interactions like the mobile menu and search filters.
